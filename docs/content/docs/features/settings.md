Configure One Calendar to match your preferences.

## Theme

Choose between Light, Dark, or System theme. The app uses `next-themes` for seamless theme switching.

## Language

Multi-language support with built-in internationalization. Select from supported languages:

- English
- Chinese (Simplified)

## Calendar Preferences

- **Default View**: Choose which view opens by default (Day, Week, Month, Year, Four-day)
- **First Day of Week**: Sunday, Monday, or Saturday
- **Time Format**: 12-hour or 24-hour

## Timezone

Select your timezone from the full IANA timezone list. Timezone-aware event display ensures correct times across regions.

## Notifications

Configure event reminders:

- Notification sound selection (multiple options)
- Custom notification times (minutes before event)
- Background notification checking every 60 seconds

## Keyboard Shortcuts

| Key | Action          |
| --- | --------------- |
| `N` | New event       |
| `/` | Search events   |
| `T` | Go to today     |
| `1` | Day view        |
| `2` | Week view       |
| `3` | Month view      |
| `4` | Year view       |
| `5` | Four-day view   |
| `←` | Previous period |
| `→` | Next period     |

Shortcuts can be disabled in settings.

## Toast Notifications

Choose toast notification position:

- Bottom-left
- Bottom-center
- Bottom-right

## Account

Manage your account:

- View profile and session info
- Sign in/out
- Account deletion
- Share management
- Import/Export tools
