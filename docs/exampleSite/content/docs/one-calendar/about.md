One Calendar is an open-source, privacy-first calendar application built with Next.js.

## Vision

Most modern calendar tools are overloaded with automation, notifications, and analytics. One Calendar takes a different approach:

- **Privacy-first**: No AI tracking, no analytics by default, no behavioral profiling
- **Local-first**: Smooth planning experience with encrypted local storage
- **Open-source**: MIT-licensed, self-hostable, community-driven
- **Simple & Clear**: Built for individuals and small teams who value clarity over complexity

## Technology Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: Zustand
- **Database**: PostgreSQL via Drizzle ORM
- **Authentication**: Better Auth
- **Email**: Resend
- **Monorepo**: Nx
