Create, edit, and organize events with a rich set of features.

## Creating Events

- **Quick create**: Click on a time slot in Day/Week view to open the event dialog pre-filled with that time
- **Keyboard shortcut**: Press `N` to create a new event at the current time
- **Event dialog**: Full-featured dialog with all event fields

## Event Properties

Each event supports:

| Property          | Description                                                |
| ----------------- | ---------------------------------------------------------- |
| Title             | Event name                                                 |
| Start/End Time    | Precise date and time controls                             |
| All-Day           | Toggle for all-day events                                  |
| Calendar/Category | Assign to a calendar category                              |
| Color             | Choose from 7 colors                                       |
| Location          | Optional location text                                     |
| Participants      | Comma-separated list                                       |
| Notification      | Reminder (at event time, 5/15/30/60 min before, or custom) |
| Description       | Multi-line text                                            |

## Editing Events

- **Inline editing**: Click an event to open the preview, then edit
- **Drag and drop**: Move events by dragging them to new times/days
- **Resize**: Events automatically update when dragged
- **Preview panel**: Quick view with edit, duplicate, share, bookmark, and delete actions

## Deleting Events

- Delete with undo support (toast notification with undo button)
- Confirmation dialog for safety
- Linked shares are automatically cleaned up on deletion

## Event Actions

- **Duplicate**: Create a copy of an event
- **Bookmark**: Save events to a bookmark list
- **Search**: Search events by title, location, or description
- **Filter**: Filter events by calendar category

## Categories

Organize events into customizable calendar categories:

- Create, edit, and delete categories
- Assign colors to categories
- Filter events by category
- Reorder categories with move up/down
- Delete category optionally removes its events
