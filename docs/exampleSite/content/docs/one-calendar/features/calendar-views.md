One Calendar provides five distinct calendar views to help you organize your schedule.

## Day View

A detailed view of a single day with hourly time slots.

- 24-hour timeline with 60px height per hour
- Drag and drop events to reschedule
- Click and drag on empty slots to create events
- Current time indicator line
- All-day events section at the top
- Auto-scrolls to current hour on open

## Week View

The default view showing a full week with seven columns.

- Configurable first day of the week (Sunday, Monday, or Saturday)
- All-day events at the top with proper stacking
- Events laid out side-by-side when overlapping
- Drag and drop events across days and times
- Click and drag on time slots to create events
- Current time indicator on today's column

## Month View

A bird's-eye view of the entire month.

- All days of the month displayed in a grid
- Up to 3 events shown per day with "+N more" overflow
- Events color-coded by category
- Previous/next month days shown in muted style

## Year View

A comprehensive yearly overview.

- All 12 months displayed in a 3-column grid
- Mini calendars for each month
- Click on any day to view events in a popover
- Responsive layout adapts to sidebar state

## Four-Day View

A compact view showing exactly four consecutive days.

- Useful for short-term planning
- Same functionality as the week view
- Ideal for weekend or focused work planning

## View Switching

Switch between views using:

- The dropdown in the top navigation bar
- Keyboard shortcuts: `1` (Day), `2` (Week), `3` (Month), `4` (Year), `5` (Four-day)
- Arrow keys to navigate between periods
